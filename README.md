# fotisdelis.com

Source for [https://fotisdelis.com](https://fotisdelis.com) — the academic site of Fotis Delis (Economist, European Commission JRC Seville).

Built with [Jekyll](https://jekyllrb.com/) using the [al-folio](https://github.com/alshedivat/al-folio) theme. Deployed via the included `.github/workflows/deploy.yml` action to the `gh-pages` branch, which GitHub Pages serves.

## Editing

- Content pages: `_pages/`
- Publications (BibTeX): `_bibliography/papers.bib`
- News items: `_news/`
- Project cards: `_projects/`

See `CLAUDE.md` in this repo for content conventions and house rules.

## Theme docs

al-folio docs are kept in repo root: `INSTALL.md`, `CUSTOMIZE.md`, `FAQ.md`, `TROUBLESHOOTING.md`, `QUICKSTART.md`, `SEO.md`, `ANALYTICS.md`. Upstream: <https://github.com/alshedivat/al-folio>.
